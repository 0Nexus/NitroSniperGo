# NitroSniperGo
Discord Nitro sniper in Go

![Screenshot](screenshot.png)

### USE :

```
 go get github.com/bwmarrin/discordgo
 go get github.com/fatih/color
 go get github.com/valyala/fasthttp
 go build
 ./NitroSniperGo
 ```
